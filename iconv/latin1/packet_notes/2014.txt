                        <b>CA3 03300 lucidum cell (LC)</b>
                </p>
                <p style="font-family: courier">
                        N = 3, one spiny and two aspiny
                </p>
                <p style="font-family: courier">
                        The 03320 class has been mapped into this 03300 class, since they differ only by dendrites in SP.
                </p>
                <p style="font-family: courier">
                        <b>New rule invoked: when compressing classes, leave axons in PCL because of the issue of terget specificity but discount dendrites. Therefore, this class is now considered 03300 (instead of 03320).</b>
                </p>

                        <b>Previously approved by DWW, CLR, CW as 03320. GAA does not need to see packet again if we agree to the rule change and approve as 03300.</b>
